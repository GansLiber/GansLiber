# 👨‍💻 Обо мне

Backend-разработчик на Go, интересуюсь высоконагруженными системами, микросервисной архитектурой и обработкой данных в реальном времени.

Мой путь в IT начался в 2020 году со стажировки на позиции Project Manager. Затем поступил на специальность «Разработчик веб и мультимедийных приложений», где изучал C, Python, JavaScript, SQL, PHP, Django, Laravel и Vue. Уже во время обучения начал работать разработчиком, постепенно полностью переключившись на backend.

В настоящее время основной стек — **Go**, **PostgreSQL**, **Redis**, **ClickHouse**, **NATS**, **Docker** и **Linux**. Интересуюсь проектированием распределённых систем, конкурентным программированием, производительностью и отказоустойчивостью сервисов.

Люблю разбираться в том, как устроены технологии "под капотом", поэтому помимо практики постоянно изучаю базы данных, брокеры сообщений, сетевые протоколы и внутреннее устройство Go.
